# ATM

> use the term for the file name

## Synonyms & Abbreviations

Automated Teller Machine, Cash Machine

## Description

An electronic telecommunications device that enables customers of financial institutions to perform financial transactions, such as cash withdrawals, deposits, or account information inquiries, at any time and without the need for direct interaction with bank staff.
