# Transaction

> use the term for the file name

## Synonyms & Abbreviations

Tx, Operation, Exchange

## Description

An instance of buying or selling something, or a business deal. In a banking context, it refers to any activity that affects an account, such as a withdrawal, deposit, or transfer, or an inquiry that interaction with the banking system.
